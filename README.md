Overview

This project is a Machine Learning-based Crop Yield Prediction System developed using Python, Pandas, Scikit-Learn, and Streamlit. The application predicts crop yield based on agricultural factors such as rainfall, temperature, and cultivation area. The goal is to assist farmers and agricultural stakeholders in making data-driven decisions to improve productivity and resource management.

Features
Predict crop yield using Machine Learning.
User-friendly web interface built with Streamlit.
Data preprocessing and model training using Pandas and Scikit-Learn.
Random Forest Regressor for accurate yield prediction.
Real-time prediction based on user inputs.
Lightweight and easy to deploy.
